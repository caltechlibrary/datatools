
# Using csv2mdtables

Convert data1.csv to data1.md using Unix pipes.

```shell
    cat data1.csv | csv2mdtable > data1.md
```

Convert data1.csv to data1.md using options.

```
    csv2mdtable -i data1.csv -o data1.md
```
